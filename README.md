
# 🚗 Car Sales Overview — Power BI

An interactive Power BI dashboard for tracking and analysing car dealership sales performance across regions, companies, body styles, and colors.


## 🎯 Objectives

- Track **YTD / MTD / YOY** metrics for Total Sales, Average Price, and Cars Sold
- Visualize sales trends by week, body style, color, dealer region, and company
- Provide a detailed transaction-level grid for drill-down analysis


## 📊 KPIs Tracked

| Category | Metrics |
|---|---|
| **Sales Overview** | YTD & MTD Total Sales, YOY Growth, PTYD Difference |
| **Average Price** | YTD & MTD Avg Price, YOY Growth, PTYD Difference |
| **Cars Sold** | YTD & MTD Cars Sold, YOY Growth, PTYD Difference |
